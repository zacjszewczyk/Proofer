Despite its immense value, though, today the internet kind of sucks. The part of the internet that most people interact with, a small subset of the entire network, has become filled with ads, trackers, viruses, spyware, and all sorts of bad actors looking to exploit naive users. The Easy Fix: Installing Software

Browser Extensions

Words: 159. Sentences: 8
Overused phrase: 0
Repeated: 0; Avoid: 0

A few key programs can stop most web-based ads, trackers, and the types of resources that load viruses and spyware. Many people work hard to exploit naive users, but many others also work hard to keep those bad actors from doing so; the latter create software like Adblock Plus, Ghostery, and uBlock Origin. Adblock Plus removes ads, which will also make websites load faster. Ghostery stops creepy websites from tracking you around the internet. uBlock Origin can target anything, making it a powerful tool for blocking whatever the other two don’t catch. Although they overlap in some areas, I prefer all three because they each specialize in something different. Installing them takes just a few clicks, and so I encourage everyone to do that right now. Even with their default settings, these tools will do a great deal to improve the average user’s web browsing experience; for even better results, spend some time learning how to fine-tune their performance.

Antivirus Software

Words: 133. Sentences: 6
Overused phrase: 0
Repeated: 0; Avoid: 0

After browser extensions, antivirus software does a great job as a second layer of defense. Windows comes with basic antivirus software by default, but those looking to beef up their security can check out some great free programs, too. Avast for all platforms, and Kaspersky for the PC, score high in all reviews. Although an easy attack vector, bad actors do not have to go through the web browser to access your machine or your network; they may target one — or both — in a variety of other ways that extensions like Adblock Plus, Ghostery, and uBlock Origin cannot stop. Should someone get into your machine or your network, Avast, Kaspersky, and similar tools will help detect and remove them. Installing these, too, takes just a few clicks; go ahead and install them now.

The Tough Fix: Altering System Behavior

Words: 130. Sentences: 7
Overused phrase: 0
Repeated: 0; Avoid: 0

Browser extensions can make it harder for bad actors to find easy ways into a target. Antivirus software adds a second layer of security by protecting the machine and the network. Good programs such as Avast and Kaspersky use passive measures like virus scans, and active measures like blocking suspicious behavior. This basis will serve the average user well. I want to stress that it does not make them impervious to all attacks, though, just immune from the most basic ways bad actors steal personal data, infect computers, and trash the web browsing experience. Again, if a smart person targets a specific computer, they will get in. Those looking for even better results do have some more advanced steps available to them, though, and so I talk about those below.

Automatic Updates

Words: 125. Sentences: 6
Overused phrase: 0
Repeated: 0; Avoid: 0

Automatic updates take too long, force reboots, and seem to happen at the worst times; turning this feature off, though, makes a hacker’s job just a little bit easier. Most system updates patch bugs and security vulnerabilities a hacker could take advantage of, so do the right thing, turn them on, and let them install often. Do the same for individual programs as well, from trusted vendors. This takes discipline, but it will make you less susceptible to the types of attacks found on public sites like Exploit Database. Exploit Database and other, similar websites list hundreds of ways a bad actor could use old versions of software and operating systems to get into your machine. Stay updated and give yourself less to worry about.

The hosts file

Words: 79. Sentences: 3
Overused phrase: 0
Repeated: 0; Avoid: 0

Another simple — albeit intimidating — way to make web browsing just a little bit better involves editing your machine’s host file. In simple terms, every time your computer tries to connect to www.google.com, it checks a system file to see whether you have visited the website before. If you have, it loads information about www.google.com from your computer and then serves the site; if not, it looks for that information on the internet, downloads it, and then serves the site.

Now imagine that instead of you trying to open Google, a website you visited tried to make a sneaky connection to www.TrackMyUsers.com. Your computer first checks the hosts file for information about that site, then the internet, and then loads the page. You can use the hosts file to fool that sleazy site, though, by giving it the wrong information about www.TrackMyUsers.com on purpose. As a result, the sleazy website does not get to track you. Privacy preserved.

Bad actors have gone to great pains to use many different websites for their sleazy purposes, but many people have gone to great pains to catalog all those pages as well. Several people publish lists with tens of thousands of websites like www.TrackMyUsers.com, with the sole purpose of helping regular users block them. I found Dan Pollock’s collection first at SomeoneWhoCares.org. After some research, I learned that Steven Black unifies fourteen different lists and posts the result for anyone to use on GitHub.

If you want to download Steven’s code, run the scripts, and set up everything on your own, go for it. On a Mac, though, you can just open Terminal, copy and paste the snippet below, hit Enter, and input your password. The snipped backs up your hosts file to hosts.bak, downloads Steven’s list, and tells your computer to read this new hosts file from then on.

sudo cp /etc/hosts /etc/hosts.bak; sudo curl https://raw.githubusercontent.com/StevenBlack/hosts/master/alternates/fakenews-gambling-porn-social/hosts -o /etc/hosts; dscacheutil -flushcache; sudo killall -HUP mDNSResponder

On Windows, open Notepad, open or create the file at C:\Windows\System32\Drivers\etc\hosts, copy and paste everything on this page into Notepad, and save it. Next, click the Start button, type “cmd”, right-click Command Prompt and choose Run as Administrator, then run the command below to tell your computer to read this new hosts file from now on:

ipconfig /flushdns
